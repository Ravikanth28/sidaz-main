## üì¶ Getting Started

### Prerequisites
- Node.js 18.x or later
- npm / yarn / pnpm

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Mordheeshvara/sidaz.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
